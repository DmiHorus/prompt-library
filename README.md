# Библиотека промптов

Командная библиотека промптов, управляемая как код. Промпты хранятся в виде YAML-карточек с метаданными, версионированием и проверкой в CI.

## Быстрый старт

1. Просмотрите промпты в `prompts/categories/`
2. Используйте шаблон YAML-карточки в `prompts/templates/card_template.yaml`
3. Перед добавлением новых промптов прочитайте [руководство по участию](docs/contribution_guide.md)

## Структура

```
prompts/
  categories/       # промпты, сгруппированные по доменам
  templates/        # базовый шаблон YAML-карточки
  variants/         # A/B и экспериментальные версии
examples/           # синтетические примеры входа/выхода
docs/               # документация и таксономия
tests/              # тест-кейсы для промптов
scripts/            # скрипты валидации и утилиты
```

## CI

Каждый PR проверяется автоматически:
- YAML lint
- Проверка обязательных полей карточки
- Поиск паттернов секретов
