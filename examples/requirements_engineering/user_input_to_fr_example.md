# Example: requirements_engineering.user_input_to_fr

## Input

```yaml
source_text: |
  «Нужно, чтобы система быстро делала отчёты и отправляла их начальству — а то сейчас всё теряется.
  Думаю, менеджеры перестанут забывать про отчётность. Зарплатные ведомости не трогаем, это в 1С.»
```
